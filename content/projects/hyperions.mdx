**Hyperions** is an API library for Hytale that is designed to provide other mods with an API for storing, transmitting, and receiving energy. The library uses the [Transfer API](https://hytalemodding.guide/en/projects/transfer-api) by community member shailist, and is based off of the [Energy](https://github.com/TechReborn/Energy) API by the TechReborn team.
